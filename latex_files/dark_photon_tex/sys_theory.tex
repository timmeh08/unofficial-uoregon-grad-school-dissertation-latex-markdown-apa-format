\subsection{Recipes of evaluating theoretical systematic uncertainties}
This section summarizes the procedures used to evaluate theoretical systematic uncertainties following the prescriptions recommended by PMG. Three main types of theoretical uncertainties are considered: renormalization and factorization scale variations, PDF variations, $\alpha_{S}$ variations.
\subsubsection{Scale Uncertainties}
Uncertainties in event yields due to renormalization ($\mu_R$) and factorization ($\mu_F$) scales are evaluated using a 7-point reweighting scheme. Individual events are reweighted with rescaled $(\mu_R, \mu_F)$ pairs according to the combinations listed in Table~\ref{tab:scalevar}, excluding those where the two scales differ by a factor of four.
The rescaled yields are computed in each analysis bin, and the asymmetric systematic uncertainty is obtained by taking the envelope of the resulting yields with respect to the nominal one:
\begin{equation}
n^{\text{up}}{\text{bin}} = \max_i \left( n^i{\text{bin}} \right), \qquad
n^{\text{down}}{\text{bin}} = \min_i \left( n^i{\text{bin}} \right)
\end{equation}

\subsubsection{PDF Uncertainties \& Alpha\_S Uncertainties}
PDF uncertainties are categorized into two types: variations within a given PDF set (intra-PDF) and differences between distinct PDF sets (inter-PDF).

\subsubsection{Intra-PDF Set Uncertainties}
PDF fits carry intrinsic uncertainties arising from the fitting procedure. As a result, each PDF set includes multiple replicas (the PDF error set), each with a slightly different parametrization. These variations must be propagated to estimate the uncertainty on the final-state observables.
The signal samples are generated using the PDF4LHC21\_40\_pdfas (LHAPDF ID 93300) set \cite{PDF4LHC}, which has 40 replicas. The background samples are generated using the NNPDF3\_nnlo\_as\_0118\_hessian (LHAPDF ID 303200)~\cite{nnpdf} set, which contains 100 replicas. For each replica, the event weights are used to compute the yield in each $m_{\mathrm{T}}$ bin. The uncertainty is then calculated as the standard deviation of the yields across all replicas:

\begin{equation}
    \Delta X = \sqrt{\frac{1}{N} \sum_i \left( X_i - X_0 \right)^2 }
\end{equation}

where $\Delta X$ represents the absolute uncertainty on the predicted yield $X$, $X_0$ is the nominal yield, and $X_i$ corresponds to the yield obtained with the $i$-th PDF replica.\\

\subsubsection{Inter-PDF Set Uncertainties}
Different PDF sets use distinct parameterizations and input datasets, resulting in differences between the central PDFs. These differences can lead to variations in final-state observables that are important to consider.
To assess uncertainties between different PDF families, for signal sample is also generated using alternative PDF sets, namely \texttt{CT18NLO} (LHAPDF ID 14400) and \texttt{MSHT20nlo\_as118} (LHAPDF ID 27100). The nominal member of each set is compared against the nominal PDF4LHC21\_40\_pdfas set to evaluate the inter-PDF variation. For background samples, alternative PDF sets \texttt{CT18NNLO} (LHAPDF ID 14100) and \texttt{MSHT20nnlo\_as118} (LHAPDF ID 27400) are used. The nominal member of each set is compared against the nominal set to evaluate the inter-PDF variation.
\subsubsection{$\alpha_{S}$ Uncertainties}
The uncertainty associated with the strong coupling constant $\alpha_S$ is evaluated by utilizing PDF sets generated with different $\alpha_S$ values for $\alpha_S = 0.117$ and $\alpha_S = 0.119$ and the uncertainty is determined using the formula:
\begin{equation}
    \Delta X_{\alpha_S} = \frac{\left| X_{\alpha_S^+} - X_{\alpha_S^-} \right|}{2}
\end{equation}
where $X_{\alpha_S^+}$ and $X_{\alpha_S^-}$ are the yields computed with the increased and decreased $\alpha_S$ values, respectively.


The final systematic uncertainty from PDF and $\alpha_S$ variations is obtained by adding the intra-PDF, inter-PDF, and $\alpha_S$ uncertainties in quadrature for each bin:

\begin{equation}
    \Delta X_{\text{total}} = \sqrt{\Delta X_{\text{intra\_PDF}}^2 + \Delta X_{\text{inter\_PDF}}^2 + \Delta X_{\alpha_S}^2}
\end{equation}
